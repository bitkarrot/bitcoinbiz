# Bitcoin Businesses Directory Worldwide

## Want to add a new entry? 

Edit this file and make a request
https://github.com/bitkarrot/bitcoinbiz/blob/main/bitcoinlinks.csv

## What

A React + TypeScript + Vite application to display Bitcoin businesses.

## Setup

clone the repository

```bash
git clone https://github.com/bitkarrot/bitcoinbiz.git
```

Build the project

```bash
pnpm install
pnpm dev
```

Download your csv file and place it in the root directory of the project, 
Name it bitcoinlinks.csv.

Get the metadata and favicons for the project and store locally to load faster

```bash
node scripts/fetch_metadata.js
node fetch_favicons.js
```

Build the project

```bash
pnpm run build
```

Deploy to vercel 

```bash
vercel --prod
```
